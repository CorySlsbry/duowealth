# DuoWealth

> Financial dashboard platform for couples budgeting & shared finance businesses.
> Built for couples and newlyweds merging finances who need real-time visibility across joint and individual accounts.

## Overview

DuoWealth is a real-time financial intelligence platform designed specifically for the couples budgeting & shared finance industry. It connects your accounting software, CRM, and operations data into one unified dashboard.

## Features

- **Real-Time Dashboard** — Live financial KPIs specific to couples budgeting & shared finance
- **AI CFO Advisor** — Ask questions and get industry-specific financial guidance
- **QuickBooks Integration** — Automatic two-way sync with your books
- **Industry Reports** — Pre-built reports tailored for couples and newlyweds merging finances who need real-time visibility across joint and individual accounts
- **Stripe Billing** — Subscription management built in

## Getting Started

1. Clone this repo
2. Copy `.env.example` to `.env.local` and fill in your keys
3. Run `npm install`
4. Run `npm run dev`
